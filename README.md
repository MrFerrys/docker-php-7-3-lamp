# docker-php-7-3-lamp
Docker container with php-7-3, apache, phpmyadmin & mariadb 

### Usage

* How to run it
```
	docker-compose -f php7-docker-compose.yml up -d --remove-orphans
```
* How to stop it
```
	docker-compose -f php7-docker-compose.yml down
```
